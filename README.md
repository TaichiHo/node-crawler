
## node-crawler

Node.js web crawler experiment


## Get Started

Run

```
npm install
```

Open browser with

```
http://localhost:3000/crawler
```

## Package Dependencies

- [node-simplecrawler](https://github.com/cgiffard/node-simplecrawler)
- [cheerio](https://github.com/cheeriojs/cheerio)
